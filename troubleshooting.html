Boot Issues

  • Stuck at Apple logo → Check GPU configuration
  • No boot picker → BIOS or OpenRuntime.efi missing

Kernel Panics

  • Wrong kext order
  • Missing VirtualSMC

No Audio

  • Wrong AppleALC layout-id
  • Audio codec not supported

No Internet

  • Missing Ethernet kext
  • Unsupported Wi-Fi card

Sleep & Wake Issues

  • USB not mapped
  • Wrong SMBIOS
