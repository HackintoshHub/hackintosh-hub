Introduction

OpenCore is a modern bootloader designed to boot macOS on non-Apple hardware with stability and security.

Requirements

  • Intel CPU (4th Gen or newer)
  • UEFI motherboard
  • 16GB USB drive

BIOS Settings

Disable Secure Boot, Fast Boot, CSM. Enable UEFI and AHCI.

Create macOS Installer

sudo /Applications/Install\ macOS.app/Contents/Resources/createinstallmedia \
--volume /Volumes/USB

EFI Folder Structure

EFI
└── OC
    ├── ACPI
    ├── Drivers
    ├── Kexts
    └── config.plist

Essential Kexts

  • Lilu.kext
  • VirtualSMC.kext
  • WhateverGreen.kext

config.plist

Edit using ProperTree and configure Kernel, NVRAM, and PlatformInfo.

SMBIOS

Use iMac19,1 or iMac20,1 for desktops.

Install macOS

  1. Boot from USB
  2. Select Install macOS
  3. Format disk as APFS

Post Installation

Copy EFI from USB to system drive.

Troubleshooting

Check boot-args, kext order, and BIOS settings.
